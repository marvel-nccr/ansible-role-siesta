# Ansible Role: marvel-nccr.siesta

An Ansible role that installs [Siesta](https://launchpad.net/siesta) on Ubuntu.

## Installation

`ansible-galaxy install marvel-nccr.siesta`

## Role Variables

See `defaults/main.yml`

## Example Playbook

```yaml
  - hosts: servers
    roles:
    - role: marvel-nccr.siesta
```
